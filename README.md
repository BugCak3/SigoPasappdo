¿Pasas? ¿No pasas? ¿Todavía hay chance? Esta página te ayuda a responderlo rápido con números reales, usando tus ponderaciones, tus notas actuales y lo que aún te falta por entregar o presentar.

https://fulanoymengano-qlapps.github.io/SigoPasappdo/

## Ejecutar en local

1. Clona o descarga este repo.
2. Abre index.html en tu navegador.

Opcional: usa Live Server en VS Code para recarga automática.

## Qué resuelve

- Organiza la materia por categorías: Trabajos, Talleres, Parciales, Semestral y las que quieras agregar.
- Deja definir el peso de cada categoría hasta completar 100%.
- Soporta varias notas por categoría (por ejemplo, 4 talleres o 2 parciales).
- Permite marcar cualquier evaluación como pendiente con Aún no la hago.
- Calcula automáticamente:
  - nota asegurada (si en lo pendiente sacas 0%),
  - nota máxima posible (si en lo pendiente sacas 100%),
  - cuánto necesitas para llegar a D, C, B o A,
  - y cuándo una meta ya no es alcanzable.

## Cómo funciona

1. Eliges categorías.
2. Asignas ponderaciones.
3. Ingresas notas y marcas pendientes.
4. Ves el resultado y los mínimos que necesitas para cada letra.

## Escala de letras

- A: 91 o más
- B: 81 a 90.9
- C: 71 a 80.9
- D: 61 a 70.9
- F: menos de 61

## Lo más útil

- Te dice si todavía puedes pasar o si ya no hay forma matemática.
- Muestra el promedio que necesitas en todo lo pendiente.
- Si tienes Parciales y Semestral pendientes, genera una tabla específica para ambos.
- Sirve para planear estrategia antes de exámenes finales.

## Stack

- HTML
- CSS
- JavaScript vanilla

Sin backend, sin base de datos y sin instalación de dependencias.


## Aviso

Los resultados son una estimación matemática basada en los datos ingresados. La nota oficial puede variar por reglas del curso: redondeos, mínimos por examen, penalizaciones o políticas internas.
